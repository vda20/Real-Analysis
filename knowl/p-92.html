Any set with two binary operations satisfying A1-A4, M1-M3, D is called a commutative ring. So the integers with the operations \(+\) and \(\cdot\) form a commutative ring. If axiom M4 is also satisfied, then \(F\) is called a field. So the rational numbers with the operations \(+\) and \(\cdot\) are an example of a field. But there are other examples. An important one is the set \({\mathbb Q}[i]=\{a+ib| a,b \in {\mathbb Q}, i^2=-1\}\text{.}\)
