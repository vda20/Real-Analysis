So the positive even integers (or positive even natural numbers) are the elements of the set
\begin{equation*} \{2,4,6,\ldots\} = \{x\in {\mathbb N}| x=2m \mbox{ for some } m\in {\mathbb N}\} \end{equation*}
and the positive odd integers are the elements of the set
\begin{equation*} \{1,3,5,\ldots\} = \{x\in {\mathbb N}| x=2m+1 \mbox{ for some } m\in {\mathbb N}\}. \end{equation*}
