We want to prove \({\cal P} \Rightarrow {\cal Q }\text{.}\) But if we start by assuming that \(n^2\) is even, that is, \(n^2=2m\) for some \(m\text{,}\) it is not clear what we should do next. It would be a lot easier if we could start with a statement about \(n\text{,}\) not \(n^2\text{.}\) Since \(\sim {\cal Q}: n \mbox{ is odd }\) and \(\sim {\cal P}: n^2 \mbox{ is odd }\text{,}\) the contrapositive says that if \(n\) is odd, then \(n^2\) is also odd. So we start by assuming that \(n\) is odd. This means that \(n=2m+1\) for some integer \(m\text{.}\) But then \(n^2=(2m+1)^2=4m^2+4m+1=2(2m^2+2m)+1\) and we can re-write \(n^2=2k+1\) where \(k=2m^2+2m\) is an integer. So we see that \(n^2\) is also odd, and we conclude that \(\sim {\cal Q} \Rightarrow \sim{\cal P }\) is true, and so \({\cal P} \Rightarrow {\cal Q }\) is also true and we are done.
