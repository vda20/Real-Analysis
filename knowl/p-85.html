We start with a set \(F\text{.}\) A binary operation on a set is any function with domain \(F\times F\) and range \(F\) (recall that \(F\times F\) is the set of all pairs \((x,y)\) of elements of \(F\)). We now assume that our set \(F\) has two binary operations, called addition and multiplication. Given an input \((x,y)\text{,}\) the output of addition is denoted by \(x+y\) and the output of multiplication is denoted by \(xy\) (or sometimes \(x\cdot y\)).
