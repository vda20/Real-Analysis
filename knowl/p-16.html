A common and useful way to describe sets is the set builder notation. Suppose \({\cal P}(x)\) is a statement depending on some variable \(x\text{.}\) We can then consider the set of all possible \(x\) that will make \({\cal P}(x)\) true, and we denote it by \(\{x| {\cal P}(x)\}\text{.}\) So if \({\cal P}(x)\text{:}\) "\(x\) is a natural number less than 5", then \(\{x| {\cal P}(x)\}=\{1,2,3,4\}\text{.}\)
