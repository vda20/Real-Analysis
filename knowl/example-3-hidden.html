Prove that if \(n\) is an even integer, then \(3n\) is divisible by 6.
In this problem, the given assumptions consist in the statement
\begin{equation*} {\cal P}: n \mbox{ is an even integer} \end{equation*}
the conclusion is the statement
\begin{equation*} {\cal Q}: 3n \mbox{ is divisible by 6} \end{equation*}
and we want to prove \({\cal P}\Rightarrow {\cal Q}\text{.}\)
So we start deriving logical conclusions. \(\cal P\) is saying that \(n\) is even, so we can write \(n=2m\) for some integer \(m\text{.}\) Then \(3n=3(2m)=6m\text{.}\) But this means that \(3n\) is divisible by 6. So we conclude that \(\cal Q\) is true.
