M1 (Commutativity)
\(xy=yx \hspace{1ex}\) for all \(x,y\in F\)
M2 (Associativity)
\(x(yz)=(xy)z \hspace{5ex}\) for all \(x,y,z \in F\)
M3 (Multiplicative identity)
There is a unique element \(1\in F\text{,}\) called the multiplicative identity, such that \(1\neq 0\) and \(1x=x \hspace{1ex} \mbox{ for all } x\in F\)
M4 (Multiplicative inverse)
For each non-zero element \(x\in F\text{,}\) there is a unique element \(x^{-1}\in F\text{,}\) called the multiplicative inverse, such that \(xx^{-1}=1\text{.}\)
