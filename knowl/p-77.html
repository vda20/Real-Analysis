If \(n\) is even, then \(n=2m\) for some integer \(m\text{,}\) and so
\begin{equation*} \frac{n(n+1)}{2}=\frac{2m(n+1)}{2}=m(n+1), \end{equation*}
an integer. If \(n\) is odd, then \(n=2m+1\) for some integer \(m\text{,}\) and then
\begin{equation*} \frac{n(n+1)}{2}=\frac{n(2m+1+1)}{2}=\frac{2n(m+1)}{2}=n(m+1), \end{equation*}
an integer. So in both cases \(\frac{n(n+1)}{2}\) is an integer, and we are done.
